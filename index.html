Análise Preditiva com Machine Learning

CLASSIFICAÇÃO AUTOMÁTICA DE TLDS USANDO K-MEANS CLUSTERING

Esse script, escrito em linguagem R, foi um desdobramento do meu projeto de iniciação científica, durante a graduação. Ele consiste na classificação de lotes de dosímetros TL de fluoreto de lítio (LiF) e sulfato de cálcio (CaSO4) irradiados com raios X no laboratório de física médica da Unicamp. A classificação foi feita com k-Means Clustering de acordo com a carga total integrada e a carga na região do maior pico de TL. O algoritmo foi avaliado através da métrica largura de forma e os melhores resultados obtidos foram 2 clusters para LiF e 3 clusters para CaSO4.

ZERO LEARNING PARA FUSÃO DE IMAGENS PET / MRI E SPECT / MRI DO CÉREBRO

Esse script, escrito em linguagem Python, consiste na aplicação de uma arquitetura CNN pré-treinada (VGG19) para fundir imagens (Lahoud & Süsstrunk, 2019). Foram usadas imagens de MRI e imagens PET ou SPECT do cérebro, de pacientes com glioma. A fusão combina a alta resolução tecidual da MRI, com funcionalidade da imagem de PET/SPECT, permitindoa localização precisa de um glioma, fundamental para um estadiamento preciso (I-IV) e para um planejamento mais acurado, em uma eventual radioterapia adjuvante (estadios II-IV).

ARQUITETURA GAN PARA AVALIAÇÃO MAMOGRÁFICA DE DENSIDADE MAMÁRIA

Esse projeto, escrito em linguagem Python, consiste na aplicação da arquitetura GAN (Generative Adversarial Network) para gerar imagens sintéticas de mamografia. A GAN foi treinada com imagens de mamografia rotuladas (com microcalcificações), em formato PGM (Portable Gray Scale). Essas imagens são comparadas com imagens sintéticas geradas por uma rede geradora (G) inicializada com um ruído. A comparação é feita por uma rede discriminadora (D) que aprende a gerar imagens bem próximas às originais.

PIPELINE DE DEEP LEARNING PARA DETECÇÃO DE ECGS ANÔMALOS USANDO AUTOENCODER

Esse projeto, escrito em linguagem Python, consiste em usar autoencoders para classificar sinais de eletrocardiogramas (ECG). Trata-se de um tipo específico de rede neural projetada para compactar dados de entrada e decodifica-los em dados de saída semelhantes. Os dados de entrada usados para treinamento foram os ritmos cardíacos normais de um paciente com insuficiência cardíaca congestiva grave. Se o sinal de saída for muito diferente do de entrada, o sinal é classificado como anômalo.
